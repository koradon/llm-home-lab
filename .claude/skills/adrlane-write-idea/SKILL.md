# Write an idea (adrlane)

Create or update a file under `docs/ideas/`.

1. Read `docs/llm/DECISION_RULES.md` (Idea section) and `docs/llm/AGENT_PROTOCOL.md`.
2. Copy `docs/llm/templates/idea.md` for new ideas.
3. Search existing ideas to avoid duplicates; prefer updating an existing file.
4. Use `## Status` and `## Related` sections — no YAML frontmatter.
5. Link related specs or roadmap items when they exist.
