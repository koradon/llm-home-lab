# M4 - Production Hardening and Multi-Node Operations

Goal: evolve the orchestrator into a dependable multi-node control plane for home-lab LLM infrastructure.

## Exit Criteria

- Multi-node registry and scheduling are stable.
- Monitoring dashboards and alerts cover critical SLOs.
- Security and access controls are implemented for API and tool usage.
- Operational runbooks exist for incident response and upgrades.
